Чтобы стартовать проект нужно:
1. Добавить новую папку mkdir project
2. Перейти в эту папку cd project
3. Создать виртуальное окружение python -m venv env
3.1 Активировать виртуальное окружение env\Scripts\activate.bat
3.2 Обновить pip python -m pip install --upgrade pip
3.3 Выполнить команду pip list для проверки 
4. Создать файл .env с переменным окружения
5. Создать файл .gitignore
6. Запустить pip freeze > requirements.txt
